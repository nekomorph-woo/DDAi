# 描述编写指南

## 核心原则

描述是 Agent 选择技能的**唯一决策依据**。

## 结构

```
[能力描述]。Use when [触发条件]。
```

## 能力描述

- 一句话概括技能功能
- 使用具体动词：提取、转换、验证、生成
- 避免模糊词：帮助、处理、支持

## 触发条件

- 列出具体关键词
- 包含文件类型
- 覆盖用户可能的表达方式

## 示例对比

| 场景 | 差 | 好 |
|------|-----|-----|
| 文档处理 | "处理文档" | "提取 PDF 文本和表格。Use when PDF、表单、文档提取。" |
| 代码生成 | "写代码" | "生成 TypeScript 类型定义。Use when 类型定义、interface、type。" |
| 数据转换 | "转换数据" | "CSV 与 JSON 互转。Use when CSV、JSON、数据格式转换。" |

## 长度限制

最多 1024 字符。超出会被截断。
